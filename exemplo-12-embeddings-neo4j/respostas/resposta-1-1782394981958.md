**O que são tensores no TensorFlow.js**

No TensorFlow.js, um *tensor* é simplesmente um array multidimensional (um vetor) cujos elementos são todos números.
Em termos de JavaScript, você pode pensar em um tensor como um array regular (`[]`) que pode conter outros arrays, formando uma estrutura em forma de tabela.

```js
// Exemplo do contexto – um tensor 2D que representa três pessoas
const tensorPessoas = [
  [30, 1, 0, 0, 1, 0], // Erick
  [25, 0, 1, 0, 0, 1], // Ana
  [40, 0, 0, 1, 1, 0]  // Carlos
];
```

Cada linha do array externo representa uma entrada (uma pessoa) e cada número dentro dessa linha é uma característica dessa pessoa após ser convertido em um valor numérico (por exemplo, idade, cor, localização, etc.).

**Por que usamos tensores**

Os algoritmos de machine learning esperam dados numéricos. Ao transformar dados do mundo real — como “idade: 30, cor: azul, localização: São Paulo” — em um tensor, obtemos uma estrutura uniforme que pode ser alimentada diretamente em uma rede neural.

**Representação em JavaScript**

- **Tensor unidimensional (vetor)** – um único array de números:
  ```js
  const vetor = [0.2, 0.8, 0.5];
  ```
- **Tensor bidimensional (matriz)** – um array de arrays, como mostrado acima.
- **Tensors de maior dimensão** seguiriam o mesmo padrão: um array de arrays de arrays, e assim por diante.

No exemplo que temos, o tensor é bidimensional porque temos várias *pessoas* (linhas) e cada pessoa tem várias *características numéricas* (colunas). Essa é a maneira que o TensorFlow.js utiliza internamente para alimentar os dados em um modelo.

**Normalização (uma dica importante)**

O contexto também menciona que, antes de criar um tensor, é comum normalizar os números para que fiquem todos entre 0 e 1. Isso ajuda o modelo a aprender de forma mais estável. No nosso exemplo, a coluna “idade” (30, 25, 40) seria escalonada para um valor entre 0 e 1, assim como as outras características codificadas (1 = presente, 0 = ausente).

**Resumo rápido**

- **Tensor = array multidimensional de números** em JavaScript.
- **Representação** → um array regular que pode conter outros arrays (por exemplo, `[[30,1,0,...], [25,0,1,...], ...]`).
- **Objetivo** → fornecer dados numéricos uniformes para algoritmos de machine learning, como redes neurais no TensorFlow.js.

Portanto, sempre que você vir um tensor no TensorFlow.js, imagine uma tabela de números — um “vetor” que pode ter qualquer número de dimensões — pronta para ser usada por um modelo para aprender padrões e fazer previsões.
