**O que é um tensor?**

No TensorFlow.js, um **tensor** é simplesmente uma estrutura de dados que contém apenas números. Pense nele como um vetor ou uma lista em JavaScript, mas com uma ou mais dimensões. Em vez de trabalhar com objetos JavaScript complexos (como `{nome: "Erick", idade: 30, cor: "azul", localizacao: "São Paulo"}`), o TensorFlow.js precisa que você converta esses dados em uma forma puramente numérica – um tensor – para que os algoritmos de machine learning possam processá-los.

**Como os tensores são representados em JavaScript**

Como o TensorFlow.js é uma biblioteca JavaScript, um tensor é criado usando a API `tf.tensor` (ou uma das suas variantes, como `tf.tensor2d`, `tf.tensor1d`, etc.). Basicamente, você fornece um array JavaScript (ou um array multidimensional) de números, e o TensorFlow.js o envolve em um objeto tensor.

```js
// Exemplo: convertendo as preferências de cores em um tensor
// Azul, Vermelho, Verde são as três classes possíveis
const coresTensor = tf.tensor2d([
  [1, 0, 0], // Erick → azul
  [0, 1, 0], // Ana   → vermelho
  [0, 0, 1]  // Carlos → verde
], [3, 3])   // forma: 3 linhas (pessoas) × 3 colunas (cores)
```

Neste snippet:

* O array externo `[[1,0,0], [0,1,0], [0,0,1]]` é um **array bidimensional** em JavaScript.
* Cada linha representa uma pessoa codificada com **one‑hot** (ou seja, um “1” para a cor escolhida e “0” para as outras).
* O tensor resultante tem a forma `[3, 3]`, o que significa “3 linhas e 3 colunas”.

**Por que usar um tensor?**

O TensorFlow.js não entende objetos JavaScript ou valores primitivos diretamente. Ao converter os dados em um tensor, você fornece ao motor de machine learning uma representação numérica uniforme que pode ser alimentada em modelos, calculada e atualizada durante o treinamento.

**Um resumo rápido**

| Conceito | Representação em JavaScript | Descrição |
|---------|------------------------|-------------|
| **Tensor** | `tf.tensor([...])` ou `tf.tensor2d([...])` | Um array de números (unidimensional, bidimensional, etc.) que o TensorFlow.js usa internamente. |
| **One‑hot encoding** | `[[1,0,0], [0,1,0], [0,0,1]]` | Uma forma de representar uma categoria usando um vetor numérico onde apenas um elemento é 1 e os demais são 0. |
| **Forma** | `[3, 3]` (linhas × colunas) | Informa ao tensor suas dimensões, o que é importante para operações matemáticas. |

**Na prática**

Quando você estiver trabalhando com dados do mundo real (por exemplo, uma lista de pessoas com várias propriedades), o fluxo típico é:

1. **Extrair** os valores numéricos que você deseja usar (por exemplo, idade, cor codificada como one‑hot, localização).
2. **Organizar** esses números em um array JavaScript (ou arrays aninhados).
3. **Criar** um tensor a partir desse array com `tf.tensor`, `tf.tensor1d`, `tf.tensor2d`, etc.
4. **Passar** o tensor para o seu modelo de machine learning.

Seguindo esses passos, você transforma dados legíveis por humanos em uma forma que o TensorFlow.js pode entender e utilizar para inferência ou treinamento.
