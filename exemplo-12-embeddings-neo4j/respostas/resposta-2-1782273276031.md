### Como converter objetos JavaScript em tensores no TensorFlow.js

TensorFlow.js trabalha com **tensors**, que são essencialmente arrays multidimensionais que armazenam apenas números.
Quando você tem um objeto JavaScript comum (por exemplo, `{ nome: "Carlos", idade: 40, cor: "verde", localizacao: "Curitiba" }`), o primeiro passo é **extrair os valores e transformá-los em um formato numérico**.

Abaixo está um guia passo a passo usando os dados que vimos no contexto.

---

#### 1️⃣ Extraia os valores brutos

```js
const pessoas = [
  { nome: "Erick",   idade: 30, cor: "azul",   localizacao: "São Paulo" },
  { nome: "Ana",     idade: 25, cor: "vermelho", localizacao: "Rio" },
  { nome: "Carlos",  idade: 40, cor: "verde",  localizacao: "Curitiba" }
];
```

A partir de cada objeto, podemos criar um array simples de números.
Para campos **numéricos** (como `idade`), usamos o valor diretamente.
Para campos **categóricos** (como `cor` ou `localizacao`), precisamos de uma representação numérica – a maneira mais comum é a **codificação one‑hot**.

```js
// 1. Mapeie cada objeto para um vetor numérico
const vetorNumerico = (p) => {
  // idade → número direto
  const idade = p.idade;

  // cor → vetor one‑hot de tamanho 3 (azul, vermelho, verde)
  const cores = ["azul", "vermelho", "verde"];
  const corIdx = cores.indexOf(p.cor);
  const corOneHot = tf.oneHot(corIdx, cores.length); // [1,0,0] etc.

  // localização → vetor one‑hot de tamanho 3 (São Paulo, Rio, Curitiba)
  const locs   = ["São Paulo", "Rio", "Curitiba"];
  const locIdx = locs.indexOf(p.localizacao);
  const locOneHot = tf.oneHot(locIdx, locs.length);

  // Combine tudo em um único vetor 1D (idade + cor + localização)
  // tf.concat combina os tensores
  return tf.concat([tf.tensor([idade]), corOneHot, locOneHot]);
};
```

#### 2️⃣ Crie um tensor 2D (uma tabela)

Agora, aplique a função de mapeamento a cada pessoa e empilhe os resultados em um tensor 2D.
`tf.stack` é útil porque ele empilha uma lista de tensores de forma uniforme.

```js
const tensors = pessoas.map(vetorNumerico);          // array de 3 tensores 1D
const tabela = tf.stack(tensors);                    // forma (3, ?)

console.log(tabela.shape);   // [3, 7] → 3 linhas (pessoas) × 7 colunas (idade + 3 cor + 3 loc)
```

**O que a tabela se parece (valores numéricos):**

| idade | cor.azul | cor.vermelho | cor.verde | loc.São Paulo | loc.Rio | loc.Curitiba |
|------|----------|--------------|-----------|---------------|---------|--------------|
| 30   | 1        | 0            | 0         | 0             | 1       | 0            |
| 25   | 0        | 1            | 0         | 1             | 0       | 0            |
| 40   | 0        | 0            | 1         | 0             | 0       | 1            |

Esta é exatamente a “tabela” que o contexto descreveu, só que expressa como um tensor do TensorFlow.js.

---

#### 3️⃣ Use o tensor para o modelo

Agora você tem um tensor 2D que pode ser alimentado diretamente em uma camada densa, uma rede recorrente ou qualquer outro operando que espere entradas numéricas.

```js
// Exemplo: camada densa simples
const modelo = tf.sequential();
modelo.add(tf.layers.dense({inputShape: [tabela.shape[1]], units: 1}));
modelo.compile({optimizer: 'sgd', loss: 'meanSquaredError'});

// Treine com os dados (apenas um exemplo, sem processamento real)
await modelo.fit(tabela, /* seu alvo */ , {epochs: 10});
```

---

### Resumo rápido

| Passo | O que fazer | Código típico |
|------|------------|----------------|
| **Extraia** | Obtenha valores brutos de cada objeto. | `p.idade`, `p.cor`, `p.localizacao` |
| **Codifique** | Converta categorias em vetores numéricos (one‑hot). | `tf.oneHot(idx, tamanho)` |
| **Combine** | Junte números e vetores one‑hot em um vetor 1D. | `tf.concat([...])` |
| **Empilhe** | Crie um tensor 2D (tabela) a partir de uma lista de vetores 1D. | `tf.stack(vetores)` |
| **Use** | Alimente o tensor em qualquer operando do modelo. | `modelo.predict(tabela)` |

---

#### 🎯 Por que isso é importante

TensorFlow.js **não entende objetos JavaScript** – ele só entende números dispostos em uma estrutura regular (um tensor). Ao converter cada objeto em um vetor numérico e, em seguida, empilhar esses vetores em um tensor 2D, você obtém exatamente o formato que o motor de machine learning espera.

A conversão também torna os dados prontos para **vetorização**, **normalização** e **augmentação**, que são etapas essenciais antes de treinar qualquer modelo.

---

### TL;DR (a versão mais curta)

1. **Extraia** os valores numéricos brutos (`idade`) e **codifique** as categorias (`cor`, `localizacao`) usando `tf.oneHot`.
2. **Combine** tudo em um vetor 1D para cada objeto.
3. **Empilhe** os vetores em um tensor 2D (`tf.stack`) – esta é a “tabela” que o TensorFlow.js pode processar.
4. **Feed** o tensor para o seu modelo.

É isso! Com esses passos, qualquer coleção de objetos JavaScript pode se tornar um tensor pronto para machine learning no TensorFlow.js. 🚀
