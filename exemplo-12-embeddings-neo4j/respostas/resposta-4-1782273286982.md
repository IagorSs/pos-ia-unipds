### Como uma rede neural funciona no TensorFlow.js

No TensorFlow.js, uma rede neural é simplesmente um conjunto de operações matemáticas interligadas que o modelo usa para descobrir padrões ocultos nos dados.
Imagine que você tem uma lista de pessoas com características como idade, cor favorita e localização. Uma rede neural vai aprender a relacionar essas características com as categorias de clientes que você já conhece (por exemplo, “premium”, “medium”, “basic”).

#### 1. Transformando dados em números
O TensorFlow.js não entende objetos JavaScript diretamente, então primeiro convertemos os dados em **tensors** — estruturas que contêm apenas números.
No exemplo, pegamos o array de pessoas e criamos uma tabela simples:

| cor     | Erick | Ana  | Carlos |
|---------|-------|------|--------|
| azul    | 1 0 0 | 0 1 0 | 0 0 1 |
| vermelho | 0 1 0 | 0 0 1 |       |
| verde   | 0 0 1 |      |       |

Isto é **codificação one‑hot**: cada cor é representada por um vetor de 0s e um único 1 que indica a cor ativa. O mesmo processo seria aplicado para idade e localização se elas fossem usadas diretamente.

#### 2. Alimentando a rede
Agora que os dados estão em forma numérica, podemos **alimentar** (ou “treinar”) a rede com dois conjuntos:

* **Características** – os tensores que representam as entradas (por exemplo, a tabela de codificação one‑hot).
* **Rótulos** – as categorias corretas que a rede deve aprender a prever (por exemplo, “premium”, “medium”, “basic”).

Durante o treinamento, a rede ajusta os pesos internos de suas operações matemáticas para que, quando uma nova entrada for fornecida, ela produza uma saída que se pareça com os rótulos corretos.

#### 3. Fazendo previsões
Depois que o modelo é treinado, você pode fornecer uma nova entrada — digamos, uma pessoa com idade 28, cor favorita “azul” e localização “São Paulo” — e a rede neural calculará a probabilidade de que essa pessoa pertença a cada categoria. A categoria com a maior probabilidade é a previsão final.

#### 4. Fluxo de trabalho típico para desenvolvedores web
Para a maioria dos desenvolvedores web, a parte pesada de ajustar os pesos do modelo é feita por engenheiros de machine learning (geralmente usando Python). Eles exportam um arquivo de modelo pronto para uso. No entanto, com o TensorFlow.js você pode:

* Carregar esse modelo pré-treinado em um aplicativo web.
* Alimentar novos dados (como os tensores que acabamos de criar) na rede.
* Obter previsões diretamente no navegador ou no lado do servidor Node.js.

#### Resumo
* Uma rede neural no TensorFlow.js é uma série de operações matemáticas que aprendem a mapear entradas numéricas para saídas desejadas.
* Primeiro, você transforma objetos JavaScript em tensores (por exemplo, usando codificação one‑hot para características como cor).
* Alimentar esses tensores em um modelo (pré-treinado ou treinado localmente) permite que você faça previsões — como classificar um novo cliente em “premium”, “medium” ou “basic”.
* Para a maioria dos aplicativos web, você geralmente carrega um modelo pronto para uso e se concentra em preparar os dados de entrada corretamente.

É assim que uma rede neural ganha vida no TensorFlow.js: dados numéricos → operações matemáticas ajustadas → previsões úteis, tudo dentro do ecossistema JavaScript.
