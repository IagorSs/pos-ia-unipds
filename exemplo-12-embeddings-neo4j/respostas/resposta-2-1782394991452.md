### Como converter objetos JavaScript em tensores no TensorFlow.js

O TensorFlow.js só consegue trabalhar com **números**, então, primeiro precisamos transformar cada objeto JavaScript em um vetor numérico.
Abaixo está um exemplo passo a passo que segue exatamente o que o contexto descreve.

---

#### 1. Comece com seus objetos

```js
const pessoas = [
  { nome: "Erick", idade: 30, cor: "azul",   localizacao: "São Paulo" },
  { nome: "Ana",   idade: 25, cor: "vermelho", localizacao: "Rio" },
  { nome: "Carlos", idade: 40, cor: "verde",  localizacao: "Curitiba" }
];
```

O TensorFlow.js não consegue usar os campos `nome`, `cor` ou `localizacao` diretamente porque são texto.
Precisamos transformá-los em números.

---

#### 2. Decida como codificar os campos categóricos

* **Campo numérico** – `idade` já é um número, podemos usá-lo diretamente.
* **Campos categóricos** – vamos usar *codificação one‑hot* para `cor` e `localizacao`.

```js
// Listas de todas as categorias que encontramos
const cores   = ["azul", "vermelho", "verde"];
const locais  = ["São Paulo", "Rio", "Curitiba"];
```

Agora, para cada pessoa, criamos um vetor numérico:

* `idade` → mantemos como está.
* `cor` → um vetor de 3 elementos onde apenas o índice correspondente à cor da pessoa é `1`, os outros são `0`.
* `localizacao` → vetor de 3 elementos semelhante para a localização.

---

#### 3. Transforme cada objeto em um vetor numérico

```js
const tensorData = pessoas.map(p => {
  // 1️⃣ idade (numérica)
  const idade = p.idade;

  // 2️⃣ um‑hot para a cor
  const corOneHot = cores.map(c => (c === p.cor ? 1 : 0));

  // 3️⃣ um‑hot para a localização
  const locOneHot = locais.map(l => (l === p.localizacao ? 1 : 0));

  // Junte tudo: [idade, cor_one‑hot..., loc_one‑hot...]
  return [idade, ...corOneHot, ...locOneHot];
});
```

Após este passo, `tensorData` se parece com:

```js
[
  [30, 1, 0, 0, 1, 0],   // Erick
  [25, 0, 1, 0, 0, 1],   // Ana
  [40, 0, 0, 1, 1, 0]    // Carlos
]
```

Cada linha agora é composta **apenas por números**.

---

#### 4. Crie um tensor do TensorFlow.js

```js
import * as tf from '@tensorflow/tfjs';

// Crie um tensor bidimensional (2D)
// - shape: [número de objetos, comprimento do vetor numérico]
// - neste caso: [3 pessoas, 6 números por pessoa]
const tensorPessoas = tf.tensor2d(tensorData, [pessoas.length, 6]);

console.log(tensorPessoas.shape);   // [3, 6]
tensorPessoas.print();
```

`tensorPessoas` agora é um **tensor de 2 dimensões** que o TensorFlow.js pode alimentar em um modelo.

---

### Por que este passo é importante

* **O TensorFlow.js só entende números.** – As operações internas do modelo são todas matemáticas; texto ou objetos brutos não podem ser usados.
* **Codificação one‑hot** – converte um valor categórico em um vetor esparso que o modelo pode interpretar como "esta característica tem este valor".
* **Tensor de 2 dimensões** – cada linha representa um exemplo (uma pessoa) e cada coluna um recurso (idade, cor, localização, etc.). Esta é a forma padrão que a maioria dos modelos de aprendizado supervisionado espera.

---

### Resumo rápido

1. **Extraia** os campos numéricos diretamente.
2. **Liste** todas as categorias exclusivas para cada campo categórico.
3. **Mapeie** cada valor categórico para um vetor one‑hot (1 no índice correspondente, 0 em todos os outros).
4. **Combine** os vetores numéricos e one‑hot em um único vetor por objeto.
5. **Use `tf.tensor2d`** (ou `tf.tensor1d` para um único exemplo) para converter o array resultante em um tensor.

Seguindo esses passos, seus objetos JavaScript estarão prontos para qualquer modelo de machine learning baseado em TensorFlow.js!
