**Treinar uma rede neural** significa ensinar o modelo a reconhecer padrões nos dados, ajustando os seus parâmetros internos até que as suas previsões fiquem cada vez mais próximas dos resultados “reais” que já conhecemos.

No exemplo que vimos, temos uma lista de pessoas com atributos como **idade**, **cor** e **localização**, e cada pessoa pertence a uma categoria de idade conhecida (jovem, adulta, idosa). Quando alimentamos essa lista na rede, os dados são primeiro convertidos em tensores e normalizados (por exemplo, a idade é transformada em um valor entre 0 e 1).

A rede começa com pesos e vieses aleatórios. Esses pesos são os números internos que cada neurônio multiplica pelos valores de entrada e soma para produzir a sua saída. A rede faz uma previsão para cada pessoa (por exemplo, “esta pessoa provavelmente pertence à categoria adulta”).

Durante o **treinamento**, comparamos a previsão da rede com a categoria real (o “rótulo” que já conhecemos). A diferença entre a previsão e o rótulo é chamada de erro. A rede então ajusta os seus pesos internos para reduzir esse erro – é como se estivesse a refinar as suas regras internas com base no feedback que recebe.

No contexto que vimos, isso significa:

1. **Entrada** – Os atributos normalizados (idade, cor, localização) são enviados para a primeira camada oculta.
2. **Processamento** – Cada neurônio na camada oculta faz os seus próprios cálculos internos, combinando os atributos de várias maneiras.
3. **Saída** – A saída da camada oculta é passada para a camada seguinte, que finalmente produz uma previsão de probabilidade para cada categoria de idade.
4. **Ajuste** – Calculamos o erro entre a previsão e a categoria real (por exemplo, “Ana tem 25 → categoria jovem”). A rede ajusta os seus pesos para que previsões futuras fiquem mais próximas dos rótulos reais.

Repetimos este ciclo muitas vezes com todo o conjunto de dados (e muitas vezes com exemplos adicionais). Com cada iteração, os pesos da rede são refinados, e as previsões tornam-se mais precisas. É por isso que dizemos que “treinamos” a rede – estamos a guiá-la, através de dados rotulados, para que aprenda os padrões subjacentes (por exemplo, como a idade se relaciona com a categoria) e possa fazer boas previsões para novas pessoas que nunca viu antes.

Em resumo, **treinar uma rede neural** é o processo de ajustar iterativamente os seus pesos internos com base no erro, até que o modelo consiga classificar ou prever com precisão novos dados, tal como aprender a distinguir entre “jovem”, “adulta” e “idosa” com base em características como idade, cor e localização.
